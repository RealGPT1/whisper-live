Whisper Live Dictate

Real-time speech transcription that runs entirely in your browser — no servers, no uploads, complete privacy.

How it works
  • Uses OpenAI's Whisper via Transformers.js — runs entirely in the browser, no server needed
  • Runs on WebGPU for fast inference, falling back to WASM
  • Live dictation always uses whisper-tiny for minimum latency
  • Voice activity detection automatically detects speech pauses
  • Drag the red marker on the audio meter to set the silence threshold
Features
  • Click any transcript item to edit or delete it
  • Import audio files (WAV, MP3, etc.) for transcription
  • Save transcripts to local storage for persistence across sessions
